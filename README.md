# Exploratory Data Analysis (EDA)

## Project Overview

This repository contains Exploratory Data Analysis (EDA) projects performed using Python.
The goal of these notebooks is to explore datasets, clean the data, visualize patterns, and extract meaningful insights.

## Repository Structure

EDA/
├── Zomato_eda.ipynb
├── Black Friday.ipynb
├── Datasets/
│   ├── zomato.csv
│   ├── Country-Code.xlsx
│   └── test.csv
│   └── train.csv
└── README.md

### Files Description

* **Zomato_eda.ipynb** – Exploratory analysis of the Zomato dataset including data cleaning, visualization, and insights.
* **Black Friday.ipynb** – Analysis of Black Friday purchase data to understand customer behavior and purchasing trends.

## Datasets

The datasets used in this project are:

* **Zomato Dataset** – Contains restaurant information such as location, ratings, and other related attributes used for analysis.
* **Black Friday Dataset** – Contains customer purchase data used to analyze buying patterns and trends during Black Friday sales.

*(Datasets may be stored in the repository or sourced from publicly available datasets used for learning and analysis.)*

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Steps in Analysis

1. Data Loading
2. Data Cleaning
3. Data Exploration
4. Data Visualization
5. Insights Generation

## How to Run the Project

1. Clone the repository
2. Install required Python libraries
3. Open the notebooks using Jupyter Notebook
4. Run the cells step by step to reproduce the analysis

## Purpose of This Repository

This repository is created for learning and practicing data analysis techniques using real-world datasets.
